🚀📚 **Project Showcase: Eduwide – Educational Management System**

I’m excited to share my latest Flutter project, **Eduwide**, a modern cross-platform **Educational Management System and Login Portal** developed as part of my **HNDIT 4232 – Mobile Application Development Assignment** at **Sri Lanka Advanced Technological Institute (SLIATE), Galle**.

🎓 **About Eduwide**

Eduwide is a cross-platform application developed using **Flutter**, designed to connect **Administrators, Teachers, and Students** through a single centralized digital platform.

The main goal of the system is to simplify and digitize educational management processes, including user management, online learning, student enrollment, class management, educational resource sharing, and payment tracking.

✨ **Key Features**

🔐 **Role-Based Authentication & Authorization**
The system provides secure login and registration with separate access levels for Admins, Teachers, and Students. Users are automatically directed to their relevant dashboard based on their assigned role.

👨‍💼 **Admin Management Portal**
Administrators can approve pending user registrations, manage student and teacher accounts, enroll students into classes, monitor online meetings, manage educational resources, and oversee payment records.

👨‍🏫 **Teacher Class Management**
Teachers can create and manage online classes, add Zoom meeting links, manage enrolled students, and monitor pending and completed payment statuses related to their classes.

📹 **Zoom Class Integration**
Teachers can create online learning sessions and share Zoom meeting links. Students can access their enrolled classes and join active online sessions directly through the application.

👨‍🎓 **Student Enrollment Management**
Administrators can enroll students into specific classes and manage student–teacher relationships through a structured enrollment system.

💳 **Payment Tracking System**
The system provides a centralized payment tracking option to monitor class fee statuses, including 🟡 Pending and 🟢 Completed payments.

📚 **Educational Resource Management**
Administrators can upload and manage learning resources, while students can access and view important educational materials through the platform.

👤 **Profile Management**
Each user role includes a dedicated profile management system where users can update personal information, academic details, subjects, streams, and profile photos.

🔥 **Real-Time Firebase Integration**
Firebase Authentication is used for secure user authentication, while Cloud Firestore provides real-time data management for users, enrollments, meetings, and payment information.

🔑 **Password Recovery**
Users can securely reset their forgotten passwords through Firebase password reset functionality.

🛠️ **Technologies Used**

📱 Flutter | 💙 Dart | 🔥 Firebase Authentication | ☁️ Cloud Firestore | 🌐 HTTP | 🕒 Timezone & Intl
